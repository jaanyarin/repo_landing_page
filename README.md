# Landing Page

Landing page con React + TypeScript + Vite + Tailwind CSS.

## Stack

| Área | Tecnología |
|------|-----------|
| UI | React |
| Lenguaje | TypeScript |
| Bundler | Vite |
| Estilos | Tailwind CSS |
| Iconos | Lucide React |
| Animaciones | Framer Motion |
| Contacto | WhatsApp |
| Hosting | GitHub Pages |
| CI/CD | GitHub Actions |

## Scripts

```bash
npm run dev     # desarrollo
npm run build   # producción
npm run preview # preview del build
```

## Despliegue

El deploy a GitHub Pages es automático con cada push a `main` vía GitHub Actions.
